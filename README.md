# 🏴‍☠️ Open Source Resistance

A single-page Jekyll minisite that hosts the Open Source Resistance manifesto at
[ossresistance.com](https://ossresistance.com).

## About

The manifesto argues that maintainers working inside companies should treat
maintenance of the open source software those companies already depend on as
part of their engineering work; without paperwork, without a branded programme
and without asking a manager's blessing. It is the next step after
[Open Source Friday](https://opensourcefriday.com) and the
[Open Source Pledge](https://opensourcepledge.com): where those ask companies
politely, the Resistance does not wait to be asked.
